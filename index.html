ШІ для вирішення задач для дошкільнят

Дана програма призначена для вирішення обмеженого класу задач з арифметики для дошкільнят.
Клас задач про стіл на якому лежать фрукти, сторонні предмети та хлопчика, який або їсть (забирає) фрукти, або навпаки кладе додатково на стіл.
Перше речення задачі - опис речей на столі.
Друге - що зробив хлопчик.
Трете, четверте і т д – питання задачі, починаються зі слова "Скільки".
